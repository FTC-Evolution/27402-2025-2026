## Overview
The FTC TeamCode for team 27402 is the code that controls our robot. It contains all Teleops and Autonomous modes.

### Key Features

- ❓ **Hmm..** - That's a great question
- 📈 **MeepMeep** - the MeepMeep robot simulation tool, is already built-in with the Decode field
- 🤖 **Autonomous** - Autonomous modes built in into classes, the classes are already separated by default
- 👽 **Ergonomic** - Teleop mode that is easy to understand and use
- 🤡 **NOT Modular** - This is **NOT** modular yet!

## Dependencies

- [Android Studio](https://developer.android.com/studio) **2024.2** or later
- [Java](https://www.oracle.com/java/technologies/downloads/) **17** or later
- [Gradle](https://gradle.org/release) **8.9** or later

## Building

### 1. Installing dependencies
Install all the dependencies listed above.

### 2. Cloning the repository
Clone the repository to your local machine. This can be done with

```fish
# Clone the repository
git clone https://github.com/FTC-Evolution/27402-2025-2026

# Go into the repository's folder
cd 27402-2025-2026
```

### 3. Building

Build the project with Gradle using the hammer icon in Android Studio.

You can install with the play/restart debug icon.
